# ❌⭕ Tic Tac Toe (Java)

A **classic Tic Tac Toe game** built in **Java**, featuring a clean interface and smooth gameplay. Designed to demonstrate core programming concepts such as logic control, event handling, and user interaction.

---

## Features

### Gameplay
- Two-player mode (Player X vs Player O)
- Turn-based system with automatic turn switching
- Win detection (rows, columns, and diagonals)
- Draw detection when no moves remain

### Controls
- Click-based grid interaction
- Automatic game state updates
- Reset functionality to start a new game

---

## How It Works

The game is implemented using **Java** with a structured, event-driven approach.

**Game Flow:**
1. Player X starts the game.
2. Players take turns selecting empty cells.
3. The system checks for a win or draw after each move.
4. The game ends when a player wins or the board is full.
5. The game can be reset to play again.
